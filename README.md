# md2sum - MD2 checksum utility

md2sum is a "md5sum"-type utility written in C that reads files passed in as arguments or standard input and outputs an MD2 checksum.

## Usage

Clone the repo, and ```make```.


Test with ```echo -n | ./md6sum``` - should output ```8350e5a3e24c153df2275c9f80692773 -```.
